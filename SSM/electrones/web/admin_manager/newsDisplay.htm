　　邵芳卿 发自福建 昨日凌晨1时以前，在伊拉克遭绑架的8位平潭人质终于回到各自家中，与苦候的亲人团聚。然而记者昨日采访得知，人质回家喜忧参半，由于债务缠身，这样的团聚在有的家庭而言已经是“团而不聚”。福建省平潭县委宣传部表示，8人一路上奔波劳累担惊受怕，过几天将安排他们集中体检并休息，并表示该县下一步将着力发展地方经济，治理整顿劳务中介组织。

　　昨天凌晨零点15分起，随着人质魏武乘坐平潭县政府的白色依维柯驰抵敖东镇敖网村家中，其余7名人质也被分别派专车送达家中。记者了解到，到家时间均未超过凌晨1时。

　　此前的近两个小时中，他们被载到平潭县政府办公楼中。在那里，根据福州地区的传统风俗，中共平潭县委书记陈承国、县长薛源官等官员亲手将热气腾腾的太平面端上8人的桌上，这也是8人回国后在家乡的土地上吃到的第一顿家乡饭。
